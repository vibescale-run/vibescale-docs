The **Task Kanban** centralizes everything you need to do on your app in one place. Tasks are automatically generated from two sources:

- **Opportunity** — actions identified by the market opportunity analysis
- **Security** — fixes recommended by the security report

You don't need to create tasks manually. VibeScale generates and prioritizes them for you.

## The 4 board columns

<CardGroup cols={2}>
  <Card title="TO DO" icon="circle">
    Tasks identified and waiting to be started. Entry point for all new actions.
  </Card>
  <Card title="IN PROGRESS" icon="circle-half-stroke">
    Tasks you have already started working on.
  </Card>
  <Card title="IN REVIEW" icon="magnifying-glass">
    Tasks completed internally and waiting for validation.
  </Card>
  <Card title="DONE" icon="circle-check">
    Finished tasks. Security tasks marked here contribute to unlocking Re-Check.
  </Card>
</CardGroup>

## Opening a task

Click any card to open the details panel. There you'll find:

- Full **title and description** of the recommended action
- **Source badge** — Opportunity or Security
- **Internal notes field** — record what was done, PR links, or context for the team
- **Save comment button** — persists your notes
- **Column selector** — moves the task to another stage
- **Complete button** — marks the task as finished and moves it to DONE

## Copying the AI prompt

Each card displays a clipboard icon. Click it to copy the **remediation prompt** generated by VibeScale. Paste it directly into your preferred AI editor — Copilot, Cursor, Bolt, or similar.

## Filtering tasks

Use the search bar at the top of the board to filter tasks by title. Filtering happens in real time.

## Security task priorities

Tasks originating from the security report have priority mapped automatically:

| Severity | Priority |
|----------|----------|
| Critical | P1 |
| High | P2 |
| Medium | P3 |

<Tip>
  Complete Critical and High security tasks to unlock the **Re-Check** in the security report.
</Tip>
