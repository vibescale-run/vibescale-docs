---
title: "Conectar o GitHub"
description: "Como autorizar o VibeScale a acessar seu repositório para executar as análises."
icon: "github"
---

A conexão com o GitHub é necessária para que o VibeScale possa escanear o código-fonte, as dependências e verificar a presença de segredos expostos no repositório. Sem essa conexão, as análises de segurança não podem ser executadas.

<Steps>
  <Step title="Iniciar a conexão">
    No wizard de criação do app, clique em **Conectar GitHub**. Você será redirecionado para a tela de autorização do GitHub.
  </Step>
  <Step title="Autorizar o VibeScale">
    Na tela do GitHub, revise as permissões solicitadas e clique em **Autorizar**. O VibeScale solicita apenas permissões de leitura — nenhuma escrita é feita no seu repositório.
  </Step>
  <Step title="Selecionar o repositório">
    Após retornar à plataforma, use o dropdown para localizar e selecionar o repositório correto. Você pode digitar para filtrar a lista caso tenha muitos repositórios.
  </Step>
  <Step title="Confirmar a instalação">
    A plataforma verifica que o acesso ao repositório está ativo. Quando confirmado, o app avança para a etapa de validação de URL e, em seguida, para o dashboard.
  </Step>
</Steps>

<Tip>
  Você pode atualizar o repositório conectado a qualquer momento diretamente pelo dashboard do app, sem precisar recriar o app do zero.
</Tip>

<Note>
  Agora, o VibeScale inclui uma verificação adicional ao reconectar um repositório para garantir que ele seja o mesmo código-base originalmente analisado. Se você tentar conectar um repositório diferente, será exibido um alerta bloqueando a ação, com opções para cancelar ou criar um novo app para o repositório diferente.
</Note>
