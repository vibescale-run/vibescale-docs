## Abril 2026

### 2026-04-22

- Adicionada verificação que bloqueia a reconexão de repositórios diferentes do original, com alerta para criar novo app


### 2026-04-21

- Atualização do relatório de segurança com nova classificação de grade, resumo executivo, categorias detalhadas e controle de Re-Check


### 2026-04-19

- Adicionado bloqueio para evitar execuções paralelas ao iniciar um novo ciclo de Opportunity Scale Review
- Re-Check: adicionada indicação visual de carregamento e notificações de erro para maior clareza ao usuário
- Melhorias no assistente de alinhamento para garantir consistência dos dados inferidos durante a validação
- Atualização do fluxo de adição de repositórios GitHub para usar deep-link na instalação existente e atualização em tempo real da lista
- Atualizado o dashboard para refletir o progresso detalhado das subetapas nas trilhas de Oportunidade e Segurança, melhorando a precisão dos indicadores de status
- Corrigido o re-run da Opportunity Scale Review para também superseder o ciclo de segurança, garantindo que o dashboard e os dados reflitam o estado correto


### 2026-04-18

- Atualizada a interface de conexão com GitHub para usar um componente de Trust Layer


### 2026-04-17

- Adicionado link para Documentação no menu de navegação do site


### 2026-04-15

- Adicionado botão para reabrir tarefas concluídas no Kanban de tarefas


### 2026-04-14

- Adicionados links para Documentação e Status no rodapé do site


### 2026-04-13

- Documentação oficial lançada em [docs.vibescale.run](https://docs.vibescale.run)
- Sistema de Pontuação de Segurança V2 disponível para todos os usuários — nova metodologia de classificação de risco com grade consolidado por app
- Badges de risco EPSS adicionados ao Relatório de Segurança — exibem a probabilidade de exploração ativa de cada vulnerabilidade
