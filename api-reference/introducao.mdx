O VibeScale **não oferece uma API pública neste momento**.

Esta seção será expandida conforme a plataforma evolui. Nas próximas versões, planejamos documentar aqui:

- Integrações com ferramentas externas
- Webhooks para eventos da plataforma
- Endpoints para automações de fluxo

Se você precisa de uma integração ou automação específica antes disso, entre em contato:

**suporte@vibescale.run**
